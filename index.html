TRABAJOS

Proyecto De Sistema de Gestión de Inventarios

Desarrollé un sistema completo de gestión de inventarios utilizando HTML, CSS y JavaScript para el frontend, y Python con Flask para el backend. El sistema permite a los usuarios agregar, actualizar y eliminar productos, así como generar informes detallados sobre el estado del inventario.

Proyecto De Sitio Web de Comercio Electrónico

Creé un sitio web de comercio electrónico responsivo utilizando HTML5, CSS3 y JavaScript. Implementé características como un carrito de compras, pasarela de pago y sistema de gestión de usuarios. El sitio está optimizado para SEO y ofrece una experiencia de usuario fluida en dispositivos móviles y de escritorio.
